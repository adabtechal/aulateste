> Quero visualizar meus leads em um quadro Kanban e configurar os stages,

## Dev Notes

### Libraries
- `@dnd-kit/core` ^6.x — Drag & drop engine

### API Contract — Move Lead
```
PATCH /api/leads/:id/stage
Body: { "stageId": "uuid" }
Response: { lead, history_entry }
```

### API Contract — Reorder Stages
```
PATCH /api/stages/reorder
Body: { "stages": [{ "id": "uuid", "position": 0 }, ...] }
Response: { stages }
```

### Tempo no Stage
Calcular a partir do último registro em `lead_stage_history` onde `to_stage_id = current_stage_id`. Usar `date-fns/formatDistanceToNow`.

### Testing
- Drag lead de "Novo Lead" para "Qualificado" → verificar `lead_stage_history` criado
- Criar novo stage → aparece no Kanban
- Excluir stage sem leads → removido
- Excluir stage com leads → bloqueado com mensagem
- Abrir 2 abas → mover lead em uma → atualiza na outra (Realtime)
