## Dev Notes

### Evolution API — Send Text
```
POST /message/sendText/{instance}
Headers: { apikey: "..." }
Body: {
  "number": "5511999999999",
  "text": "Olá, tudo bem?",
  "delay": 1000,
  "linkPreview": false
}
```

### Evolution API — Send Media
```
POST /message/sendMedia/{instance}
Headers: { apikey: "..." }
Body: {
  "number": "5511999999999",
  "mediatype": "image",
  "mimetype": "image/png",
  "caption": "Confira nossa proposta",
  "media": "https://example.com/image.png",
  "fileName": "proposta.png"
}
```

### Message Log Insert
```sql
INSERT INTO message_log (lead_id, direction, message_type, content, media_url, whatsapp_message_id, status)
VALUES (uuid, 'outgoing', 'text', 'mensagem', null, 'wamid.xxx', 'sent')
```

### Connection Check Flow
1. Fetch active instance from `whatsapp_instances`
2. Check `GET /instance/connectionState/{instance}`
3. If `state !== 'open'` → show error: "WhatsApp não está conectado. Vá para Configurações > WhatsApp para conectar."

### Testing
- Enviar texto para número válido → mensagem entregue + registrada no log
- Enviar imagem com legenda → imagem entregue + log com media_url
- Enviar com instância desconectada → erro informativo
- Histórico mostra mensagens em ordem cronológica
