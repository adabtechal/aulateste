## Dev Notes

### API Contract — List Messages
```
GET /api/messages?lead=uuid&type=text&direction=incoming&from=2026-03-01&to=2026-03-31&page=1&limit=50
Response: {
  data: [{
    id, lead_id, lead_name, lead_phone, direction, message_type,
    content, media_url, status, sent_at
  }],
  pagination: { page, limit, total, totalPages }
}
```

### API Contract — Export CSV
```
GET /api/messages/export?lead=uuid&direction=auto&from=2026-03-01
Response Headers: Content-Type: text/csv, Content-Disposition: attachment; filename=messages-export.csv
Response Body: CSV data
```

### Unified Timeline Query
```javascript
// Fetch both sources
const [messages, stageHistory] = await Promise.all([
  supabase.from('message_log').select('*').eq('lead_id', leadId),
  supabase.from('lead_stage_history')
    .select('*, from_stage:kanban_stages!from_stage_id(name), to_stage:kanban_stages!to_stage_id(name)')
    .eq('lead_id', leadId)
]);

// Merge and sort
const timeline = [
  ...messages.data.map(m => ({ ...m, type: 'message', timestamp: m.sent_at })),
  ...stageHistory.data.map(h => ({ ...h, type: 'stage_change', timestamp: h.moved_at }))
].sort((a, b) => new Date(b.timestamp) - new Date(a.timestamp));
```
